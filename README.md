# 📁 My Developer Portfolio
This is the source code for my professional portfolio website. It serves as a central hub for my projects, technical skills, and my journey into Software Engineering.

## 🚀 Live Link
**See it in action:** [https://ollie-dev.netlify.app](https://ollie-dev.netlify.app)

---

## 🛠️ Built With
This project was developed using a mobile-first approach to ensure a seamless experience across all devices.
* **HTML5 & CSS3:** For semantic structure and custom styling.
* **Bootstrap 5:** Used for the responsive grid system and navigation.
* **AOS (Animate On Scroll):** For smooth visual transitions.
* **Netlify:** For continuous deployment and hosting.

---

## 📸 Key Features
* **Project Gallery:** Showcasing live deployments like *Gracie Bites* and *GreenRoot*.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Contact Integration:** Linked to my professional Instagram and GitHub profiles.
